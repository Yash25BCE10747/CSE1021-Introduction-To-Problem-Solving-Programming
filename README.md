Online Shopping Cart System

About the Project

This is a simple Python program that works like a small online shopping cart.
Users can see products, add items to the cart, remove items, and check the total price.
This project is made for practicing basic Python programming.

Features

Show list of products
Add product to cart
Remove product from cart
View cart and total amount
Exit the program


Requirements

Python 3 installed on your computer


How to Run

1. Open the folder where the file is saved.
2. Open Command Prompt / Terminal in that folder.
3. Run this command:



python code.py

How to Use

Enter the number from the menu.
Follow the instructions shown on the screen.
Add or remove products using product ID.


File Name

code.py – main program
